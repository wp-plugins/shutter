== Description ==

A WordPress plugin specifically for photographers.

Available Shortcodes:
`[shutter_gallery name="Example Gallery"]`

== Installation ==

1. Install Shutter either via the WordPress.org plugin directory, or by uploading the files to your server.
2. After activating Shutter, you will see the Shutter menu item from your WordPress Dashboard.
5. You're all set. Now start showing off your pictures!
